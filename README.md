# PRODIGY_WD_05
Weather App — Live Weather Using OpenWeather API

This Weather App is a simple, clean, and responsive web application that allows users to fetch real-time weather data based on their current location or any city name entered manually. It uses the OpenWeatherMap API to provide temperature, humidity, wind speed, weather conditions, and dynamic weather icons.
<br>
Features

• Fetch weather automatically using user’s geolocation
• Enter a city manually to get weather
• Real-time temperature, humidity, wind speed
• Weather condition icons
• Loading and error handling
• Fully responsive UI
• Clean gradient interface<br>

Tech Stack

• HTML5
• CSS3
• JavaScript (Fetch API + Geolocation API)
• OpenWeatherMap API<br>

Project Structure

index.html
style.css
script.js
README.md<br>

How to Run

Download or clone the repository
git clone https://github.com/MehreenKhan465/PRODIGY_WD_05.git<br>

Open the folder

Insert your own OpenWeatherMap API key inside script.js

Open index.html in your browser
That’s it — your Weather App is ready!<br>

App Logic Overview

• Uses Geolocation API to auto-detect user weather
• If location denied, prompts manual input
• Fetches weather using OpenWeatherMap REST API
• Dynamically updates DOM elements
• Displays loading state + meaningful errors<br>
